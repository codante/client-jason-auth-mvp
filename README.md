# 报表台骨架

一个基于 Streamlit 的简洁报表台应用。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

应用会自动在浏览器中打开（通常是 http://localhost:8501）

## 功能说明

- 📁 **文件上传**：支持 CSV 和 Excel (.xlsx) 格式
- 📊 **数据总览**：显示总记录数、总列数、数值列平均值
- 📉 **趋势图**：自动识别日期列和数值列，生成趋势图
- 📥 **导出报表**：占位功能（待开发）

## 使用示例

1. 在左侧边栏上传一个包含数据的 CSV 或 Excel 文件
2. 查看数据总览卡片
3. 查看自动生成的趋势图
4. 点击底部"导出报表"按钮（目前显示 TODO）
